# Glass Effect Parameters

| Parameter           | Default | Range    |
| ------------------- | ------- | -------- |
| displacementMap     | ""      | string   |
| displacementScale   | 65      | 10 - 120 |
| glassBlur           | 25      | 8 - 40   |
| glassSaturation     | 185     | 0 - 400  |
| glassBrightness     | 60      | 0 - 500  |
| glassContrast       | 120     | 0 - 500  |
| highlightReflection | 0.15    | 0 - 1.0  |

Pass via `:user-options` prop to override defaults.
